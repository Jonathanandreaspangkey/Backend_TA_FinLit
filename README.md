## Description
This is the backend codebase of **FinLit**, a web-based financial education application designed to help users understand financial concepts in an interactive and fun way.

This backend is built using **Spring Boot** and connected to a **PostgreSQL** database to manage user data, learning modules, materials, quizzes, and user learning progress.

This application supports:
- Management of modules, submodules, materials, and quizzes, pretests, posttests
- Tracking progress of materials and quiz results and overall progress percentage (Users)
- Integration with React-based frontend
